# VIDEOJUEGO PARA LA ASIGNATURA DE PROGRAMACIÓN MULTIMEDIA
### Álvaro Vicario y Eduardo Liñares

El fin de este proyecto es crear un pequeño videojuego basándonos en el mítico "Frogger". 

Para ello, nos planteamos varios objetivos:
- Mostrar un pequeño escenario.
- En ese escenario, mostrar los "obstáculos" en movimiento.
- Mostrar a nuestro personaje, y otorgarle la capacidad de moverse según dónde el usuario toque la pantalla.
- Si se produce una colisión con un obstáculo, el jugador pierde (pierde puntos) y su posición se resetea.
- Por el contrario, si toca una moneda, gana "X" puntos.
- El objetivo principal. Si el jugador llega al final del camino, gana la partida y su puntuación se almacena.
