# VIDEOJUEGO PARA LA ASIGNATURA DE PROGRAMACIÓN MULTIMEDIA
### Álvaro Vicario y Eduardo Liñares

El objetivo de este proyecto es crear un pequeño videojuego basándonos en el género "_match 3_". Utilizaremos parte de las ideas propuestas en el conocido juego "Candy Crush" como base e inspiración directa, dentro de lo posible.
